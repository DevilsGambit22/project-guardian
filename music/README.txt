PROJECT GUARDIAN MUSIC FOLDER

GitHub Pages automatic detection:
1. Add any number of .mp3 files to this music folder.
2. Commit and push them to the public GitHub repository.
3. The project automatically detects every MP3 through the GitHub Contents API.
4. No JavaScript filename list needs to be edited.
5. Tracks play in the order returned by GitHub and loop continuously.

Local fallback:
For local testing without GitHub Pages, name tracks:
track1.mp3
track2.mp3
track3.mp3
...
up to track30.mp3

Browser rule:
Music begins only after the visitor presses AWAKEN GUARDIAN because browsers block autoplay before user interaction.

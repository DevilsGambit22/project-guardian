# Project Guardian v2.0 — Pixel Castle Prototype

Open `index.html` through GitHub Pages or a local web server.

## Included
- 1:44 awaken/patrol/return lifecycle
- Sanctuary, Grand Hall, Courtyard, and Rooftop scene rotation
- Guardian invocation, walking, running, combat, idle, and return animations
- RPG dialogue portraits and state-based messages
- Procedural ambience, particle effects, responsive Chess.com-friendly layout

## Important
This prototype stores its state only in the current browser session. True synchronized world state for all visitors requires a hosted JSON/API endpoint or another shared backend.

## Blue Knight vs. Red Knight combat
The Blue Guardian now encounters a hostile Red Knight during patrol. Both knights enter the Grand Hall, charge, exchange attacks, and the Blue Guardian defeats the enemy before resuming her watch.

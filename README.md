# Project Guardian v2.0 — Cinematic Forum Experience

## Features
- Larger pixel Guardian with stable nearest-neighbor rendering
- Cinematic eclipse, ruins, banners, stone floor, fog, embers, and summon circle
- Spawn sequence with title reveal and generated browser audio
- Accelerated cursor/touch pursuit instead of simple sliding
- Three randomized attack patterns
- Gold impact bursts, screen flash, and camera shake
- Decorative Guardian Integrity bar
- Return-to-post behavior when the hunt ends
- Responsive desktop/mobile layout
- No external libraries or sound files required

## Publish
Upload every file and folder to the root of the `project-guardian` repository, then enable GitHub Pages from the main branch root.

## Chess.com embed
Use your GitHub Pages URL in an iframe. A height around 950–1100 pixels works well for the complete experience.

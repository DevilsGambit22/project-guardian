# Project Guardian v1.0

Standalone ACFA forum experience.

## Publish
1. Upload all contents to a GitHub repository.
2. Enable GitHub Pages from the main branch/root.
3. Embed the published URL into a Chess.com forum iframe.

## Controls
- Awaken Guardian: starts a 15-second cursor hunt.
- End Hunt: immediately stops it.
- Test Hurt: previews the hurt animation.
- Mobile users tap the arena to move the target.

## Important
The supplied sprite artwork came from user-provided source images. Confirm that you have permission or an appropriate license before publishing it publicly.

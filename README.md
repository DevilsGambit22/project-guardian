# Project Guardian v1.2 — Musical Hunt

## New in this version

- The Guardian remains active for **1 minute 44 seconds (104 seconds)**.
- Clicking **Awaken Guardian** starts an original Project Guardian background theme.
- The soundtrack fades in during awakening.
- A visible countdown appears beside the status while hunting.
- When the timer expires—or **End Hunt** is pressed—the Guardian returns to post.
- The music fades out during the return sequence, resets, and remains off while dormant.
- The Guardian disappears after reaching post and stays invisible until awakened again.

## Music file

The included soundtrack is located at:

`assets/audio/guardian-theme.mp3`

You can replace it with another MP3 using the same filename. The page will continue to control it automatically.
